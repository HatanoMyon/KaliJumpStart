# KaliJumpStart
Personal kali linux jump start script to prepare a fresh installation for pentest work.
Intended for use in VM.

## Instructions:

1. sudo su root
2. cd ~
3. git clone https://github.com/HatanoMyon/KaliJumpStart (manual type)
4. cd KaliJumpStart
5. chmod +x ./kjs.sh
6. ./kjs.sh
7. Follow prompts
8. Reboot
9. Profit!
