# KaliJumpStart
Personal kali linux jump start script to prepare a fresh installation for pentest work.
Intended for use in VM.

## Instructions:

1. sudo su root
2. git clone https://github.com/HatanoMyon/KaliJumpStart (manual type)
3. cd KaliJumpStart
4. chmod +x ./kjs.sh
5. ./kjs.sh
6. Follow prompts
7. Reboot
8. Profit!
